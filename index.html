‹ Steps checklist
Manage Battery Settings
In Settings, search “Battery” and go to “Battery Optimisation” > All apps and search StepSetGo > Don't allow
Allow Autostart
In Settings, search “App Launch” and go to “App Launch”. Turn switch to 'Manually' and keep all switches enabled.
Lock StepSetGo as recent app
Recent Apps > Hold the StepSetGo app card and pull down
You could also try:
Disable Power Genie
Security > Apps > If Power Genie is installed, disable it.
Allow running in the background
Settings > Apps > StepSetGo > Battery > Enable “Keep running after screen off”, disable “Power-intensive prompt”
Remove Restrictions
Settings > Apps > Advanced > Ignore Optimisations > Allowed > All Apps > Set StepSetGo to Allowed
Alternatively, try this:
Settings > Apps > Settings > Special Access > Ignore Battery Optimisation > Select Allow for StepSetGo
Alternatively, try this:
Settings > Apps > Advanced > Ignore Optimisations > Allowed > All Apps > Set StepSetGo to Allowed
Adjust Power Plan
Security > Advanced Settings > Battery Manager > Set Power plan to Performance, set StepSetGo as Protected app
Alternatively, try this:
Settings > Advanced Settings > Battery Manager > Protected Apps > StepSetGo needs to be protected
Allow background activity
Launch StepSetGo > Press the phone’s hardware app list button (square icon) > Find the StepSetGo card > Press the lock icon next to the name to show LOCKED...or swipe down the StepSetGo card to lock (lock icon should appear next to the App name)
